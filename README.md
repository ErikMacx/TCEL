# Eric McLean / TCEL website

Marketing site for Eric McLean (TCEL, The Consulting and Education Link): strategy
and transformation for the CEOs of complex, multi-business groups in the Gulf and
EMEA. Built for findability by AI answer engines: clean static HTML, answer-first
content, structured data, and a crawlable, well-mapped site.

## Stack

- **Astro** (static output) with `@astrojs/sitemap`.
- **Cloudflare Pages** for hosting (auto-deploy on push to `github.com/erikmacx`).
- Domain **tcel.com** (Namecheap), canonical host `https://www.tcel.com`.

## Develop

```bash
npm install
npm run dev        # http://localhost:4321
npm run build      # static output to dist/
npm run preview    # serve the built dist/
```

Drafts (all seeded Insights briefs are `draft: true`) are visible in `npm run dev`
and hidden in `npm run build`.

│   │   └── insights/
│   │       ├── index.astro   Insights hub, list driven by the collection
│   │       └── [slug].astro  one answer-first page per brief (Article + FAQPage schema)

## AI findability

- `robots.txt` welcomes all crawlers, naming AI agents (GPTBot, ClaudeBot,
  PerplexityBot, Google-Extended, CCBot, Bytespider, Applebot-Extended).
- `llms.txt` is a short factual map: who Eric is, what he does, frameworks, regions, links.
- Sitemap generated by `@astrojs/sitemap` at `/sitemap-index.xml`.
- Schema: Person + ProfessionalService site-wide, FAQPage on the homepage,
  Article + FAQPage on each brief.

## Content

Insights briefs are markdown in `src/content/briefs/`. Frontmatter:
`title` (the question), `dek` (answer-first summary), `theme`
(Strategy | AI | Leadership | SME | Gulf), `date`, `readingTime`, `draft`,
optional `featured` and `comingSoon`. The hub groups briefs by theme, newest first.
